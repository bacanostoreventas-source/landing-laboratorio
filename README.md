## Cómo agregar la portada del ebook

1. Guarda tu imagen como `img/portada.png`
2. En `index.html`, busca el comentario `<!-- BOOK COVER —`
3. Descomenta la etiqueta `<img>` y elimina el div `.book-placeholder`

## Despliegue

Esta landing es un archivo HTML estático. Puedes subirla a:
- **Netlify**: Arrastra la carpeta al dashboard
- **GitHub Pages**: Activa Pages en el repo
- **Hostinger / Namecheap**: Sube via FTP
